AI Prompt → FlowGraph JSON
Copy this prompt into Copilot/AI. Then provide your flowchart content (from PDF/Visio) as the input.
Tip: ask the AI to output JSON only (no markdown) to avoid parse errors.
Small example that is compatible with this app’s loader.

      


      

        
Required node fields

        
{
  "id": "string (unique)",
  "name": "string",
  "owner": "string (lane/group)",
  "description": "string",
  "dependencies": ["id-of-prereq-1", "id-of-prereq-2"]
}

        
Rules

        
    
          
  • Output must be a JSON array of node objects: [{...},{...}]
    • 
          
  • Dependencies are prerequisites: if B requires A, then B.dependencies includes A.
    • 
          
  • All dependency IDs must exist in the same output (no missing references).
    • 
          
  • No duplicate IDs.
    • 
          
  • If the diagram contains a loop, add a review/approval/checkpoint node to break the cycle.
    • 
        

      


      

        
    
          
  1. Start from your PDF/Visio flowchart.
    2. 
          
  2. Extract steps and arrows (or describe lanes/steps) and paste into Copilot/AI along with the Prompt tab content.
    3. 
          
  3. AI outputs JSON array only.
    4. 
          
  4. Load the JSON with Browse JSON File.
    5. 
          
  5. Verify nodes/edges and adjust as needed.
    6. 
        

        
Common failure: JSON loads but edges missing → dependency IDs don’t match node IDs exactly.
